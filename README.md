# Azure DevOps CI/CD Project

## Overview
This project demonstrates a basic DevOps workflow using:
- Python Flask application
- Docker containerization
- GitHub Actions CI/CD pipeline

## Features
- Simple web application built with Flask
- Dockerized for consistent deployment
- Automated build pipeline triggered on push to main

## Tech Stack
- Python
- Flask
- Docker
- GitHub Actions

## Project Structure
- `app.py` - Flask application
- `requirements.txt` - Python dependencies
- `Dockerfile` - Container build instructions
- `.github/workflows/ci-cd.yml` - CI/CD workflow

## Outcome
This project showcases hands-on experience with CI/CD automation, containerization, and DevOps fundamentals.
